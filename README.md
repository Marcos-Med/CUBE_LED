# Cubo de LED 8X8X8

Este projeto foi desenvolvido durante o curso técnico em Eletrônica, com o objetivo de criar um sistema capaz de reproduzir animações luminosas tridimensionais utilizando um microcontrolador Arduino Uno. O cubo é composto por 8 LEDs em cada eixo (largura, comprimento e altura), totalizando 512 LEDs distribuídos em uma matriz 8×8×8.
